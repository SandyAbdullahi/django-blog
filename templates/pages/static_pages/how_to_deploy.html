How to Deploy a Django App to Heroku

|Deploying your Django app to Heroku can be quite daunting when you’re a beginner, especially when you don’t understand what some of the addons required (e.g. Dj-database-url) do to help configure your app for Heroku. In order to understand how this method works and what the various modules that are imported into your app do. We need to go through a step by step process of deployment, where at each step I will explain why we install certain addons and we create files that run specific commands.

Before starting, you need to sign up for a Heroku account. If you already have one then confirm that you have the Heroku CLI Installed on your machine. In order to do that, you must have git installed first because Heroku relies on Github to stage your changes and any updates that you commit and push to your repository. So if you already have Git installed then we can move on to step 1, other wise you need to go here.
